# lambodar-creation

Lambodar Creation - Premium Fashion Ecommerce Website

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand (State Management)
- Dukaan.io API Integration

## Features

- Premium fashion brand design
- Responsive mobile-first layout
- Product catalog with filters
- Cart management
- Checkout with Dukaan integration
- Legal pages (Privacy, T&C, Shipping, Returns)
